## 📖 sign the guestbook

the guestbook on the page is plumbed straight into **this repo's Issues**.

1. on the site, type your message in the form at the bottom of the page
2. click **♥ sign ♥** → opens a pre-filled new issue here
3. click **Submit new issue** on github
4. your message appears on the page next time anyone reloads

your github username + avatar show up next to your comment, in an animated 5-stop gradient color uniquely derived from `sha-256(your-username)` ✿ — every guest gets their own personalized rainbow.

> ### [🪐 leave a guestbook entry now →](https://github.com/luksgrin/luksgrin.github.io/issues/new?title=%5Bguestbook%5D+anonymous&labels=guestbook)

## 🐛 found a bug?

open an issue *without* the `guestbook` label and the maintainer will see it.  
PRs are welcome — but please don't try to make it look modern. that defeats the entire point.

## 🛠 the stack

- pure html / css / js — no bundler, no framework, no build step
- github pages for hosting
- guestbook → github issues rest api
- visitor counter → [moe-counter](https://github.com/journey-ad/Moe-Counter)
- music player → youtube iframe player api *(with shuffle on)*
- ~50 inline svgs drawn by hand, all images self-hosted in `/images`
